Sum of All Fears

Given a list of numbers and a number K, return whether any two numbers from the list add up to K.

  • The basic challenge of Sum of All Fears is to write a program that uses a hard-coded array of [10, 15, 3, 7] and allows the user to enter a value for K before running the algorithm and return true or false.
  • Technologies Used: HTML, JavaScript, jQuery, CSS, Bootstrap
